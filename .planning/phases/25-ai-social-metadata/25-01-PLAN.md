    - `async generateMetadata({ transcript, platform, tone, client }): Promise<Metadata>` — extract text (throw typed `EmptyTranscriptError` if empty), build prompts, call client, JSON.parse, sanitize; on JSON.parse/zod failure do ONE corrective retry via client (append "Devolvé SOLO el JSON válido…"), else throw MetadataValidationError.
