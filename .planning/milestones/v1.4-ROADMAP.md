# Milestone v1.4: Studio como producto usable

**Status:** ✅ SHIPPED 2026-06-06

## Overview

Convert the Studio from a style editor into an end-to-end usable product: one click
generates the video with live progress, configs are reusable as named profiles, social
metadata is AI-generated from subtitles, and the interface is matured toward the chosen
north-star. Frontend tooling non-negotiable: every studio-facing phase invoked
`impeccable` + `frontend-design`.

## Key Accomplishments

- **Render-from-button with live progress (Phase 23):** "Render Video" → `POST /api/render` → api-server `/batch` (BullMQ); ~1.5s status poll with Spanish step labels + percent; success/failure surfaced inline; finished MP4 preview/download proxied from `quality-finalizer/output.mp4`. Font resilience: local-first → gstatic-retry → bundled-sans chain with vendored woff2, so a transient gstatic blip can't abort a 666-frame render.
- **Named config profiles (Phase 24):** save/load/list/rename/delete named profiles; atomic writes; persist across Docker rebuilds (PROFILES_DIR bind-mount). Apply writes the active config so the editor restores state.
- **AI social metadata (Phase 25):** generate title/description/hashtags from the Whisper transcript via the local-llms router (OpenAI surface, gpt-oss:120b default); "Metadata de redes" panel replaces the Phase-22 placeholder; per-field inline edit + copy; regenerate by tone/platform without re-running the pipeline.
- **UI convergence (Phase 26):** north-star visual maturity pass — color law D-04 (blue selection / single green Render CTA), shared z-ladder, OKLCH token discipline across 6+ components, preset cards (011-C/014-C), responsive reflow (018-B). Rail (033) + header consolidation (037) deliberately deferred to Phase 27+.
- **Post-milestone hardening (this session):** profiles gained JSON export/import (portable backup/share), F5-persistence fixes, a persisted active-profile pointer with "Modificado" divergence marker + mini-specimens, and `PUT /api/config` now syncs the active profile. Production tunnel moved to HTTP/2 (stable). Verified via 21/21 browser E2E + round-trip.
