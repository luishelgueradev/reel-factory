3. Implement `remapTimestamps(originalTimeMs: number, silenceCuts: SilenceCutList): number` per D-02:
   - This is a FULL TIMELINE REMAP function, not per-word subtraction
   - Uses binary search through silence cuts (sorted by original_start) to find the applicable cumulative_shift
   - Algorithm: Find the last silence cut where `original_start <= originalTimeMs / 1000`. If found, new time = `originalTimeMs - silenceCuts.cuts[i].cumulative_shift * 1000`. If not found (time is before any silence cut), return original time unchanged.
   - Edge case per CONTEXT specifics: If silence was removed from the beginning, the first word's timestamp may shift to near-zero. The function must handle this correctly by applying the cumulative_shift.
   - Also handle the edge case where no silence cuts apply (original time before first cut) — return unchanged time.
   
   ```typescript
   export function remapTimestamps(originalTimeMs: number, silenceCuts: SilenceCutList): number {
     const originalTimeSec = originalTimeMs / 1000;
     const cuts = silenceCuts.cuts;
     
     // Binary search for the last cut where original_start <= originalTimeSec
     let left = 0;
     let right = cuts.length;
     while (left < right) {
       const mid = Math.floor((left + right) / 2);
       if (cuts[mid].original_start <= originalTimeSec) {
         left = mid + 1;
       } else {
         right = mid;
       }
     }
     
     // left - 1 is the index of the last cut that starts before originalTimeSec
     const applicableCutIndex = left - 1;
     if (applicableCutIndex < 0) {
       return originalTimeMs;  // Before any silence cut
     }
     
     return originalTimeMs - Math.round(cuts[applicableCutIndex].cumulative_shift * 1000);
   }
   ```

   ```typescript
   export function remapWordTimestamps(
     words: WhisperWord[],
     silenceCuts: SilenceCutList | null
   ): WhisperWord[] {
     if (!silenceCuts || silenceCuts.cuts.length === 0) {
       return words;  // D-03: graceful handling — use original timestamps
     }
     return words.map((w) => ({
       ...w,
       start: remapTimestamps(w.start * 1000, silenceCuts) / 1000,
       end: remapTimestamps(w.end * 1000, silenceCuts) / 1000,
     }));
   }
   ```
