## Component Specifications

### 1. Bold / Italic Toggle Row (new â€” Subtitles tab + Titles tab)

Pattern: two-button segmented toggle, identical to the existing Highlight Transition
(Fade / Instant) control in StyleControls.tsx.

**Bold row (Subtitles tab):**
- Label: `Font Weight` at 13px, color #bbb
- Two buttons: `Regular` (fontWeight=false â†’ 400) and `Bold` (fontWeight=true â†’ 700)
- Selected state: border `1px solid #4CAF50`, background `rgba(76, 175, 80, 0.12)`, color `#a5d6a7`
- Unselected state: border `1px solid #444`, background `#2a2a3e`, color `#ccc`
- Default: Bold selected (D-03 preserves existing hardcoded 700 behavior)

**Italic row (Subtitles tab):**
- Label: `Font Style` at 13px, color #bbb
- Two buttons: `Normal` (fontStyle=false) and `Italic` (fontStyle=true)
- Same selected/unselected visual as Bold row
- Default: Normal selected

**Titles tab:** Same Bold + Italic rows added inside the add/edit form for title styles,
positioned between the font family selects and the line height / padding row.

### 2. Font Size Slider â€” Extended Range (Subtitles tab)

Existing control in StyleControls.tsx: `min=24 max=120`.
Change: `max={200}`. Label and value display unchanged.
Endpoint hints: update from `<span>120</span>` to `<span>200</span>`.

**Titles tab (TitleEditor.tsx):**
- Title font size slider: current `min=24 max=120` â†’ `min=24 max=200`
- Subtitle font size slider: current `min=16 max=80` â†’ `min=16 max=200`

### 3. Outer Glow Section Card (new â€” Subtitles tab + Titles tab)

Implemented as a collapsible section card matching the Background Highlight card pattern:
- Outer container: `padding: "12px 16px"`, `background: "#1e1e2e"`, `borderRadius: 8`,
  `border: "1px solid #333"`
- Header row: `<input type="checkbox">` + label `Outer Glow` at 14px weight 600 color #e0e0e0
- When enabled=true, reveal three controls indented at `marginLeft: 24`:

  **Glow Color:** `<input type="color">` â€” same ColorControl pattern (48Ă—36px swatch).
  Label: `Glow Color` at 12px color #999. Hex display at 11px color #666 beneath swatch.
  Default value: `#ffffff`.

  **Intensity slider:** `min=0 max=1 step=0.05`. Label: `Intensity: {value}` at 12px #999.
  Endpoint hints: `0` / `1`. Default: 0.8. Represents alpha multiplier applied to glow color.

  **Softness slider:** `min=0 max=60 step=1`. Label: `Softness: {value}px` at 12px #999.
  Endpoint hints: `0px` / `60px`. Default: 20. Maps to CSS `text-shadow` blur radius.

All three sliders use `accentColor: "#4CAF50"`.

### 4. Plus Jakarta Sans â€” Font Selector (Subtitles tab)

Changes required:
- `AVAILABLE_FONTS` array: add `"PlusJakartaSans"` at index 0 (displayed first).

## CSS Implementation Reference

### Outer Glow CSS

```
text-shadow: 0 0 {softness}px {colorWithAlpha}
```

Where `colorWithAlpha` is the glow `color` with its RGBA alpha set to `intensity`.
Example: color `#ffffff`, intensity `0.8`, softness `20` â†’
`text-shadow: 0 0 20px rgba(255, 255, 255, 0.8)`

Single CSS `text-shadow` layer. Multi-layer stacking is left to Claude's discretion at
execute time (permitted per CONTEXT.md discretion note) â€” use a single layer as the baseline.

Glow applies on top of the existing `textShadow` field (not a replacement). If both
`textShadow` and `outerGlow` are enabled simultaneously, both `text-shadow` declarations
are combined into a comma-separated value.

### fontWeight mapping

```
config.fontWeight === true  â†’ CSS font-weight: 700
config.fontWeight === false (or undefined, default true) â†’ CSS font-weight: 700
```

Default is `true` (bold) to preserve existing behavior. `TikTokLayout.tsx` currently
hardcodes `fontWeight: 700` â€” must be changed to read `config.fontWeight !== false ? 700 : 400`.

### fontStyle mapping

```
config.fontStyle === true  â†’ CSS font-style: italic
config.fontStyle === false (or undefined) â†’ CSS font-style: normal
```
