| Docker image rebuild | Image rebuild must NOT affect the host-side pipeline/ directory |

| T-17-07 | Tampering | docker compose build clobbering active config | accept | The active config is on the host bind mount (./pipeline/), not inside the image. docker compose build touches only the image layer — it cannot modify the host filesystem. Verified by the rebuild test in Task 1. |
| T-17-08 | Information Disclosure | Checkpoint human-verify exposes config contents in terminal | accept | The config contains only styling data (no secrets, no PII). Viewing it in terminal output is safe for a single-developer local workflow. |

</threat_model>
