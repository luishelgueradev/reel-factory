<objective>
Fix the double-remap bug that causes subtitle sync drift and broken word-by-word coloring.

Root cause: process.sh runs Whisper on the already-cut video (silence-cutter → whisper), so
timestamps in transcript.json are ALREADY on the silence-removed timeline. remapTimestamps in
captions.ts then subtracts silence cuts AGAIN, causing progressive drift and corrupting
createTikTokStyleCaptions grouping (tokens where fromMs > toMs).

Purpose: Restore word-by-word subtitle sync and highlighting — the killer feature.
Output: Detection function + auto-skip logic + unit tests + render logging.
</objective>
