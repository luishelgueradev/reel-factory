Add `JumpCutTransition` component at the composition root. Ensure subtitles and titles are outside the zoom container (D-10).
- **Files:** `services/remotion-renderer/src/Root.tsx`
- **Requirement:** D-08, D-09, D-10
- **Detail:**
  1. Update `RemotionProps` interface to add:

     ```typescript
     import type { TransitionEvent } from "./compositions/JumpCutTransition";

     export interface RemotionProps {
       videoSrc: string;
       captionPages: TikTokPage[];
       videoWidth?: number;
       videoHeight?: number;
       totalDurationMs?: number;
       subtitleLayout?: SubtitleLayoutMode;
       subtitleConfig?: SubtitleConfig;
       titles?: TitleConfig[];
       // Phase 7: Visual effects (D-08, D-10)
       zoomEvents?: ZoomEvent[];           // Computed in render.ts
       transitionEvents?: TransitionEvent[]; // Computed in render.ts
     }
     ```
  2. Update `SubtitledVideo` component to implement D-10 visual order:
     ```tsx
     const SubtitledVideo: React.FC<RemotionProps> = ({
       videoSrc,
       captionPages,
       subtitleConfig,
       titles,
       zoomEvents = [],
       transitionEvents = [],
     }) => {
       // ...existing config merging...
       return (
         <AbsoluteFill style={{ backgroundColor: "#000" }}>
           {/* D-08: ZoomContainer wraps OffthreadVideo — subtitles/titles NOT zoomed (D-10) */}
           <ZoomContainer zoomEvents={zoomEvents} totalDurationMs={totalDurationMs ?? 10000}>
             {videoSrc && <OffthreadVideo src={staticFile(videoSrc)} />}
           </ZoomContainer>
           {/* Subtitles on top of video — not affected by zoom */}
           <SubtitleLayoutRenderer captionPages={captionPages} config={config} />
           {/* Title overlays on top of subtitles — not affected by zoom */}
           {(titles ?? []).map((title, i) => {
             const fps = 30;
             const fromFrame = Math.round(title.startTimeMs * (fps / 1000));
             const durationInFrames = Math.max(1, Math.round(title.durationMs * (fps / 1000)));
             return (
               <Sequence key={`title-${i}`} from={fromFrame} durationInFrames={durationInFrames}>
                 <TitleOverlay
                   text={title.text}
                   subtitle={title.subtitle}
                   style={title.style}
                   durationMs={title.durationMs}
                   fontFamily={config.fontFamily}
                 />
               </Sequence>
             );
           })}
           {/* D-09: Jump-cut transitions at silence cut boundaries */}
           {transitionEvents.length > 0 && (
             <JumpCutTransition
               transitionEvents={transitionEvents}
               totalDurationMs={totalDurationMs ?? 10000}
             />
           )}
         </AbsoluteFill>
       );
     };
     ```
  3. Update the `RemotionRoot` default props to include empty arrays:
     ```typescript
     defaultProps={{
       videoSrc: "",
       captionPages: [] as TikTokPage[],
       videoWidth: 1080,
       videoHeight: 1920,
       totalDurationMs: 10000,
       subtitleLayout: "tiktok" as SubtitleLayoutMode,
       subtitleConfig: { ... },
       titles: [] as TitleConfig[],
       zoomEvents: [] as ZoomEvent[],
       transitionEvents: [] as TransitionEvent[],
     }}
     ```
  4. The absolute layering ensures `ZoomContainer` only scales the video feed. Subtitles render on top at fixed position. Titles render above subtitles. Transitions render as the topmost overlay. This matches D-10 exactly.
