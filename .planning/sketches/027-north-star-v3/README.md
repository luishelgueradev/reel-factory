# Sketch 027: North-Star v3 (live preview folded in)

## Design Question
The current canonical screen (023-B) was composed **before** 025 retired the static subtitle
specimens. It still shows a frozen caption. This sketch folds the **live word-by-word caption
preview** (025-C: specimen loops the *style*, stage transport scrubs the *real moment*) into the
full canonical composite and checks two things:

1. **Coherence** — does the animated preview still read as the same calm pro panel, with the
   4-tab bar (Video pushed right), list-forward Overlays (019-C), font sheet (016-C), and timeline
   strip (020-C) all composed?
2. **The two-animations risk** — 025 judged "specimen loop + stage transport playing at once" an
   *acceptable* competing-motion risk **in isolation**. Does it still hold in the full dense
   composite, or does the live composite need a focus discipline?

## How to View
open .planning/sketches/027-north-star-v3/index.html

The subtitles tab is open by default so both animations are visible immediately. Press ▶ in the
stage transport and watch the panel specimen at the same time.

## Variants
- **A: Todo encendido** — the maximal north-star v3: live preview **+** awake per-platform metadata
  column (026-C) **+** timeline open (020-C) **+** Render available (010-A) **+** drag-on-preview
  (007). The complete picture once every post-016 sketch ships.
- **B: Corte comprometido + preview vivo** ★ (default) — the honest update to the canonical: 023-B's
  ship-first slice (timeline collapsed, Render ghosted, click-to-select, metadata "Próximamente")
  **but the caption now plays**. Specimen loops the style, transport scrubs the moment. Both run
  simultaneously, as 025-C committed to.
- **C: + disciplina de una animación a la vez** — same committed slice, but resolves the competing
  -motion risk: when you play the **stage** transport, the panel specimen **pauses itself**, so only
  one word-by-word animation moves at a time. Tests whether the full composite needs this rule.

## What to Look For
- Does the moving caption pull your eye away from the controls, or does the calm 90ms color
  transition keep it ambient? (The whole point of "the tool disappears into the task.")
- In **B**, with both the stage and the specimen highlighting words, do they fight for attention?
  Switch to **C** — does the auto-pause feel like relief, or like the panel went dead?
- In **A**, does the awake metadata column (per-platform TikTok/Reels/Shorts caption + char count +
  hashtags, with **Regenerar/Copiar in accent-blue, never action-green**) earn its 304px now that
  it's no longer "Próximamente"?
- Does the 4-tab bar + live preview + timeline still feel like **one** tool, or has v3 crossed into
  "video editor" territory the product doesn't want yet?
